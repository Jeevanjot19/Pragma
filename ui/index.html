Pragma: The Complete GTM System
Pragma is Blostem's end-to-end GTM intelligence platform, combining three layers:
📚 Layer Overview
  • WHO Layer: AI-discovers fintech prospects using news monitoring, funding signals, and hiring patterns. Scores prospects by commercial signal strength.
  • WHEN Layer: Intelligent temporal scoring that identifies the optimal time to reach each prospect based on event recency and engagement signals.
  • HOW Layer: Generates 3 persona-specific emails (CTO, Business Contact, CFO) with compliance checks and revenue proof calculations for each prospect.
  • ACTIVATE Layer: Post-signature partner activation tracking that detects stalls using real API call data and generates targeted interventions.
    • 🎯 The Demo Flow
    Start with WHO Pipeline to see discovered prospects ranked by signal strength → check WHEN Priorities to see this week's best timing → generate HOW Outreach to see personalized 3-persona campaigns → review ACTIVATE Dashboard to see post-signature partner health
